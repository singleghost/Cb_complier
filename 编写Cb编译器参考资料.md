# 编写Cb编译器参考资料

一般编程语言的语法单位有下面这些：

* 定义（definition）
* 声明（declaration）
* 语句（statement）
* 表达式（expression）
* 项（term）

“定义”是指变量定义、函数定义或类定义等

函数或方法的定义的本体中包含有“语句”，如 if 语句，while 语句，for 语句等

“表达式”是比语句小、具有值得语法单位。具有值，是指将表达式写在赋值的右侧，或者在函数调用时写在参数的位置等。if、while、for 是语句，所以不具有值。

项是表达式中构成二元运算的一方，也就是仅由一元运算符构成的语法。

定义包含语句，语句包含表达式，表达式包含项。
